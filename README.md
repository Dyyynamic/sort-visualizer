# Sort Visualizer

Inspired by Timo Bingmann's [15 Sorting Algorithms in 6 Minutes](https://www.youtube.com/watch?v=kPRA0W1kECg).

## Running

Install SFML:

```bash
# Debian/Ubuntu
sudo apt install libsfml-dev
# Fedora
sudo dnf install SFML-devel
# Arch
sudo pacman -S sfml
```

Compile and run:

```bash
make && ./sort
```
